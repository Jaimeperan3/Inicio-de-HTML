# 00 ESTRUCTURA BASICA PAGINA WEB

Toda página web tendrá al menos las siguientes etiquetas

```html

<!-- Indica el lenguaje usado en el archivo -->
><!DOCTYPE html>

<!-- Indicamos el inicio del codigo html y el idioma del contenido -->
<html lang="es">

<!-- Aqui pondre los metadatos -->
    <head>
        <!-- Indico el juego de caracteres -->
        <meta charset="UTF-8">

        <!-- ??? -->
        <meta name="viewport" content="width=device-width, initial-scale=1.0">

        <!-- Nombre pestaña superior  -->
        <title>Nombre de la pagina</title>
    </head>


    <!-- Todo lo que este en el body aparece en pantalla -->
    <body>
    
        <h1> Hola web inicial </h1>
    </body>
</html>
